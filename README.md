# SK11 Digital Release Promo Page

A digital release promo page for the SK_ELEVEN label where users can **listen to tracks on SoundCloud, submit feedback to unlock downloads, and access WAV/MP3 links after submission**. This is an **interactive marketing/release page** for a music label.

---

## Features

- **Embedded SoundCloud players** for streaming individual tracks  
- **Interactive feedback form** where users submit their thoughts to unlock downloads  
- **WAV/MP3 download links** provided after submission  
- **Responsive layout** that works across desktop and mobile devices  
- **Custom fonts and styling** for a professional music label aesthetic  

---

## Tech Stack

- **HTML** – structured content and media integration  
- **CSS** – responsive design, custom fonts, and layout  
- **Form handling** – simple submission via Netlify forms  

---

## Notes

- Designed as an interactive **promo/marketing page** rather than a full web application  
- Demonstrates the ability to integrate **media, forms, and responsive design** in a real-world music release scenario
